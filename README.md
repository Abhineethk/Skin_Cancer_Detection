# Skin Cancer Detection

This project uses a Convolutional Neural Network (CNN) to detect skin cancer from image data.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Google Colab

## Description
The model is trained on image datasets to classify different types of skin cancer. 
Preprocessing techniques such as resizing, normalization, and augmentation are applied to improve performance.

## Features
- Image classification using CNN
- Data preprocessing and augmentation
- Model evaluation using accuracy metrics
